# Learnspot

Learnspot is a FastAPI-based program with the goal to help learn German.
It provides help with translating words or texts, writing and cleaning your writings in German. It has an AI-chatbot, which try its best to focus on your learning and correcting mistakes.
What is more, it provides small tests based on user's input for practicing

# Tools

FastAPI, SQLAlchemy, Pydantic, OpenAI, Docker, Reverso.net (used for translating words)
